「Learn how to learn」笔记

发表于 2017-08-06

这门课程内容很多，但对我个人来说有启发作用对是下面这些内容。

基础

我们对大脑有两种思维模式：专注与发散
专注模式是集中精力与目标，而发散模式则是迂回试探对靠近目标。两种模式都很重要，共同形成神经通路。

拖延症的根本原因及一种解决方法
在接触新事物的时候，大脑就会产生不快感，为了逃避这种不快感，大脑自然就会去寻找更容易的事物。这就是产生拖延症的根本原因。我们对抗它的方法也很简单——面对新事物的时候，先靠自己的意志力克服这种不快，一心一意的专注于此，然后休息一小段时间，让发散模式进行处理。

我们的记忆：工作记忆与长期记忆
工作记忆也就是我们常说的“短期记忆”，每次只能缓存四个知识块。
工作记忆转变为长期记忆需要不断的间隔重复，不断巩固神经通路，并不是密集型的记忆，而是要间隔重复的记忆。因为在重复的间隔期内，发散模式在起作用，形成神经通路。

睡眠很重要
充足的睡眠会清除白天大脑产生的毒素，重演白天的记忆，巩固神经通路。

组块化

组块化是将一类碎片收集起来，将相同的知识组成一个小知识块。组块化能够使知识更容易管理，不至于零散，像拼图一样嵌入已有的知识拼图。

建立chunk的方法
1、全神贯注的关注需要组块的信息上，杜绝其他干扰。
2、理解问题。当你能够单独完成问题的时候才算真正的掌握。
3、获取背景知识，不断重复相关或不相关的问题。

一些注意事项
1、不要仅仅反复阅读不过脑子，使用回顾法效果更好。
2、不要对自己的学习能力感到自信，有时候觉得自己理解了，实际上并没有，要经过实践检验。
3、自测、试错更重要。
